# Todo App

Простое Flutter-приложение для списка задач.

## Функции
- Добавление задачи (заголовок, описание)
- Просмотр списка задач
- Удаление задачи
- Уникальный ID для каждой задачи (генерируется из времени)

## Как запустить
1. Установите Flutter
2. Клонируйте репозиторий
3. Выполните `flutter pub get`
4. Запустите `flutter run`
